The official SiteGrapher WordPress plugin for page view analytics.

== Description ==

This plugin allows site owners to easily integrate SiteGrapher into their Wordpress site without having to change any source code or theme files.

This enables you to monitor your Wordpress blog's traffic in real-time!

To use SiteGrapher on your blog, simply download this plugin and sign up for an account at https://www.sitegrapher.com.

== Installation ==

1. Sign up for a SiteGrapher account at http://www.sitegrapher.com
2. Upload the directory called `sitegrapher` to `/wp-content/plugins/`
3. Activate the plugin
4.  Go to the plugin's Settings page and enter your 16 digit site key, which can be found here http://www.sitegrapher.com/wordpress.

== Frequently Asked Questions ==

= Do I need a SiteGrapher account for this Plugin to work? =
You'll need to sign up for SiteGrapher at http://www.sitegrapher.com to make use of this plugin. If you're already using SiteGrapher, you'll simply need your 16 digit site key (which can be found here http://www.sitegrapher.com/wordpress) to make use of this plugin.
